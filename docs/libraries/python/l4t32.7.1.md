## Python 3.8.0

To get this version of python, just:

```bash
sudo apt-get update && apt-get upgrade -y
```

Then:

```bash
sudo apt-get install -y python3.8 python3-pip && apt-get clean
```

To use it type:

```bash
python3.8
```
